# DataProvider

数据管理类是 DataProvider，命名空间：ZM\Utils\DataProvider。方法均为静态方法。此组件用于操作文件类缓存、Json 对象以及资源目录和反代链接的类。

# getResourceFolder()

获取资源文件夹目录。

# addSaveBuffer()

同注解 @SaveBuffer 的作用。

# saveBuffer()

保存所有自动保存的缓存变量到文件
